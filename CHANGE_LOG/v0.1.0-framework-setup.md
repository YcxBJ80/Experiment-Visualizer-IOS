# v0.1.0 - 基础框架搭建

**日期**: 2025-12-06

## 概述

搭建了跨平台（iOS / macOS）SwiftUI 应用的基础框架，采用简洁 ChatGPT 风格界面，核心功能为全屏 WKWebView 渲染大模型生成的 HTML 可视化内容。

---

## 新增文件

### 1. `Theme.swift`
定义应用的统一配色与样式：
- **背景色**: `#0F1A1B`（深色）
- **主按钮色**: `#6ECBD3`（青色）
- **输入框背景**: `#1E2A2C`
- **文字颜色**: 白色及 60% 透明度白色
- **AccentButtonStyle**: 统一的按钮样式

### 2. `Models/ChatMessage.swift`
消息数据模型，为后续对话流预留：
- `MessageRole`: 用户 / 助手
- `ContentType`: 纯文本 / HTML
- `ChatMessage`: 包含 id、角色、内容类型、内容、时间戳

### 3. `WebView.swift`
跨平台 WKWebView 封装：
- 支持 iOS（UIViewRepresentable）和 macOS（NSViewRepresentable）
- 加载 HTML 字符串，自动包装深色背景样式
- 透明背景，与应用主题融合

---

## 修改文件

### `ContentView.swift`
重构为简洁 ChatGPT 风格界面：
- **全屏 WebView**: 展示 HTML 可视化内容，填满整个屏幕
- **底部悬浮输入栏**: 
  - 简洁圆角文本框
  - 圆形发送按钮（`#6ECBD3`）
  - 轻量阴影背景，不割裂主视图
- **占位函数**: `sendMessage()` 预留 OpenRouter API 接口
- **示例 HTML**: 居中欢迎信息，便于预览

---

## 技术要点

1. **跨平台兼容**: 使用条件编译 `#if os(iOS)` / `#elseif os(macOS)` 处理平台差异
2. **极简设计**: 单一纵向布局，避免容器分割，保持轻量留白
3. **深色主题**: 统一 `#0F1A1B` 背景贯穿 Native UI 与 WebView
4. **响应式 WebView**: HTML 内嵌 viewport 设置，适配不同屏幕尺寸
